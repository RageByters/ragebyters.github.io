## Pull Request Process

1. **One feature per PR** - Don't bundle multiple changes
2. **Describe your changes** - What and why
3. **Test everything** - Make sure it works
4. **Be responsive** - Reply to review comments
5. **Be patient** - We'll review as soon as possible

## Game Ideas We're Looking For

- Classic arcade games (Pac-Man, Space Invaders, etc.)
- Puzzle games (2048, Match-3, Sudoku)
- Reaction games (Flappy Bird, etc.)
- Original ideas!
